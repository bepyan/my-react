### 🎨 Atomic guidline

- Atoms: 기본적인 Element 단위.
- Molecules: 원자를 엮어 조금 복잡한 단위이다.
- Organisms: 유기체는 분자와 원자의 위치값을 조정한다.
- Templates: 페이지의 그리드를 정해준다.
- Pages: 페이지의 state관리 및 템플릿에게 실제 데이터를 제공해준다.
